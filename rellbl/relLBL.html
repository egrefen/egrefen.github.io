class relLBL.relLBL.Model(betas, data_vars, rel_vocab, ent_vocab, rel_unigram_probs, ent_unigram_probs, embedding_dim=50)¶

A simple relLBL model.

__init__(betas, data_vars, rel_vocab, ent_vocab, rel_unigram_probs, ent_unigram_probs, embedding_dim=50)¶

Intialise a relLBL model.

Parameters:
  • betas (theano.tensor.TensorType) – Symbolic variable for latent variables (betas).
  • data_vars (3-tuple of theano.tensor.TensorType) – 3-tuple containing symbolic variables dataset relations, entitites, and directions.
  • rel_vocab (numpy.ndarray) – Vocabulary embedding matrix for relations.
  • ent_vocab (numpy.ndarray) – Vocabulary embedding matrix for entities.
  • rel_unigram_probs (numpy.ndarray) – Vector of unigram probabilities for relations.
  • ent_unigram_probs (numpy.ndarray) – Vector of unigram probabilities for entities.
  • embedding_dim (int) – Dimensionality of word/entity/relation/etc. embeddings.
