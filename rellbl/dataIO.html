DataIO¶

Classes and functions¶

rellbl.dataIO.queries_to_matrix(relations_dict_source, entities_dict_source, queries_source, output_filename)¶

Load dataset of queries into numpy array.

Parameters:
  • relations_dict_source (string) – Path to pickled dictionary mapping relations to indices.
  • entities_dict_source (string) – Path to pickled dictionary mapping entities to indices.
  • queries_source (string) – Path to queries file, with each line in the format ‘rel.r ent.e.[12]’.
  • output_filename (string) – Path where output file should be saved.
rellbl.dataIO.vocab_txt_to_matrix(vocab_dict_source, vocab_source, output_filename)¶

Load a vocab matrix in text form in a numpy array and save it.

Parameters:
  • vocab_dict_source (string) – Path to pickled vocab to index dictionary.
  • vocab_source (string) – Path to vocab file where the each line has a vocab word, followed by a list of numbers.
  • output_filename (string) – Path where output file should be saved.
rellbl.dataIO.vocab_txt_to_tensor(vocab_and_class_dict_source, vocab_source, num_classes, class_size, output_filename)¶

Load a vocab matrix in text form in a numpy tensor and save it.

Parameters:
  • vocab_and_class_dict_source (string) – Path to pickled vocab to (word index, class index) dictionary.
  • vocab_source (string) – Path to vocab file where the each line has a vocab word, followed by a list of numbers.
  • num_classes (int) – Number of classes produced by clustering algorithm
  • class_size (int) – Maximum number of words per class.
  • output_filename (string) – Path where output file should be saved.
