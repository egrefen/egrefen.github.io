Utilities¶

Classes and functions¶

rellbl.utilities.load_data(source, type=<type 'int'>)¶

Loads array-like data from text file into numpy array.

Parameters:
  • source (string) – Path to data text file.
  • type (type) – Type of the data in text file.
rellbl.utilities.prep_gpu_data(data, borrow=True, int_cast=False, name=None)¶

Puts input array in shared theano variable, casting if necessary.

Parameters:
  • data (Numeric array-like) – Array of data to be used.
  • borrow (bool) – borrow parameter for theano.shared
  • int_cast (bool) – Theano shared variable will be tensor.cast to int32 if true.,
  • name (string) – Name of object variable.
rellbl.utilities.random_mat(dim1, dim2=None, spread=2)¶

Generate a random matrix.

Parameters:
  • dim1 (int) – Number of rows.
  • dim2 (int) – Number of columns (dim1 used if not specified).
  • spread (int or float) – Values of matrix are bounded by +/- the spread.
rellbl.utilities.random_vec(dim, spread=2)¶

Generate a random vector.

Parameters:
  • dim (int) – Dimensions of vector.
  • spread (int or float) – Values of vector are bounded by +/- the spread.
